# jslib
Javascript libraries

## Utils
- DOM: DOM manipulation
- TYPE: DataType manipulation
- AJAX: AJAX helpers (GET, POST, DELETE, PUT)
- MATH: Math functions
- PATH: Path utils
- URI: Uri helper functions

## Components
- FORM: A collection of form component: floating label, selector, slider...
